## 🤖 Project 2 — AI Data Classification

This project demonstrates how AI can be used to classify unstructured product data into structured categories.

### 📌 Objective
To organize raw product names into consistent and meaningful categories using AI assistance.

### ⚙️ Process
- Input raw product names into Excel
- Use ChatGPT to classify each product into predefined categories
- Ensure category consistency and accuracy
- Organize results into a structured table

### 📂 Categories Used
- Food
- Fashion
- Electronics
- Skincare
- Others

### 🧰 Tools Used
- Microsoft Excel
- ChatGPT (AI)

### 📊 Result
Unstructured product data was successfully classified into consistent categories, improving data organization and usability.

### 💡 Insight
This project shows how AI can speed up data processing tasks and improve efficiency in data entry workflows.
